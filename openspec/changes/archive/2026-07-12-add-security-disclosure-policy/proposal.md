## Why

ChangeRail is public source and includes agent tooling, runtime records and
supply-chain configuration. Public users need a tracked policy for reporting
suspected vulnerabilities without publishing sensitive details.

## What Changes

- Add `SECURITY.md`.
- Link the security policy from public docs.
- Extend release discipline requirements to include security disclosure policy
  checks.
- Verify the final tracked payload with strengthened public-safety scans.

## Capabilities

### New Capabilities
- none

### Modified Capabilities
- `changerail-release-discipline`: public release discipline includes tracked
  security disclosure policy and public-safety verification.

## Impact

- `SECURITY.md`
- `README.md`
